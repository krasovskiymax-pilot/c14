# Шаблон описания релиза для GitHub Release

Скопируйте и вставьте в поле "Describe this release", заменив `X.Y.Z` на актуальную версию:

---

## ChatList X.Y.Z

Сравнение ответов нейросетей — отправка одного промта в несколько моделей ИИ (OpenAI, Claude, Llama и др. через OpenRouter).

### Установка

Скачайте `ChatList-X.Y.Z-setup.exe` и запустите установщик.

### Требования

- Windows 10/11 (64-bit)
- Файл `.env` с API-ключами (см. `.env.example`)

### Возможности

- Выбор и сохранение промтов
- Улучшение промтов с помощью ИИ
- Сохранение выбранных ответов в базу
- Настройки темы и шрифта
